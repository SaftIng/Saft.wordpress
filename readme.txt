== Description ==

This plugin contains the Saft library (+ vendors) and provides a Wordpress integration. The Saft library is the aim to build a collection of components which helps anyone who wants to create applications by using RDF/Semantic Web technology. That includes node handling, store access (Virtuoso, HTTP), query caching and REST interface. Saft provides an integration layer to enable the developer to use the other RDF frameworks all together. The reason behind that is, that for instance, ARC2 and Erfurt framework are not under active development. As a developer you will be able to use all that mature frameworks inside of Saft.

**Notice:** The Saft library uses vendors with different licenses. They are considered to be align with the GPL 2 license or later.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. After the plugin is activated, you should be able to use Saft components inside your code.
